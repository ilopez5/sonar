# SONAR

## A custom benchmark for I/O systems

### Configurable parameters:
- \# of I/O dumps
- Size of I/O request
- Workload type:
  - Read-only
  - Write-only
  - Read with compute
  - Write with compute
  - Mixed w/o compute
  - Mixed with compute
- Access Pattern:
  - Sequential
  - Random
  - Strided
    - Stride length
- Compute Intensity
  - None
  - Sleep-simulated
  - Simple
  - Intense
